This is just a repo where i designed my profile's README. 
If you wish to do the same follow these steps:
## 1. Create a new repo

## 2. Name it (your name) like this: `Username / Username`

## 3. Keep the visibility Public.

## 4. Make sure "add README" is checked ON.

## 5. After creation:
Now that you've created the repo, any changes you make to the readme within it, will appear on your profile's frontpage.
